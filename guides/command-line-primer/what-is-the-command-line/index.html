What Is It

CLI means Command Line Interface, or simply Command Line.

The command line is usually a black screen or window into which you can enter and run plain text commands.

This means you need to know about the commands that are available and how to run them. There are no buttons, no icons and no menus you can use to navigate.

Long ago the command prompt was the only way to use your computer.

Photo Sai Kiran Anagani

Why To Learn

The command line offers unrivaled power, not available from most graphical programs.

  • Command line knowledge is often expected
  • You can perform advanced actions that most graphical programs cannot
    • You can process hundreds or thousands of files at once
    • You can automate common tasks in customizable ways
  • You have more flexibility and control over how you build your C++ programs
  • You gain indepence from and more freedom of choice in your other tools
  • Graphical programs often hide important information
  • You can more easily diagnose and fix various problems
  • Knowing the command line is important for every programmer

What Is It Good For

Some examples of things you could do

  • Backup your vacation photos
  • Backup your game saves
  • Run different Minecraft versions with different mods
  • Launch a video game with a high priority for better performance
  • Compile and run a program you wrote
  • Compile and run a program someone else wrote
  • Manage a Git repository
  • Rename hundreds of files at once
  • Resize, rotate, crop and blur hundreds of images at once

What Is A Shell

A shell is a program that lets you run commands from the command line.

A shell also lets you write and run your own programs in the form of shell scripts.

What Is A Terminal

A terminal is a program that lets you run a shell.

Actually Two Programs

Using the command line usually looks like using a single program but it’s actually two programs, your terminal and your shell.

Your terminal gives you a window and it runs your shell. Your shell then asks you to enter and run commands.

You can think of this as a single program for practical purposes.

These two programs, acting as one, are often referred to as the “command line”, “command line interface” or CLI for short.

Someone may also say “command prompt”, because your shell will ask (or prompt) you to enter commands.

Terminal Programs

Linux, Mac and Windows all have terminal programs available by default.

Windows

The new Windows Terminal is much better than the older versions.

Download the new Windows Terminal.

Alternatively try ConEmu instead.

Mac

Mac has apps like the built in Terminal.app and also 3rd party apps like iTerm.

Linux

Linux has many, including xterm, aterm, konsole, gnome-terminal, lxde-terminal etc. One or more of these will be already installed by default on most Linux systems.

Shells

Bash

Bash is the Bourne Again SHell, one of the most common shells, though there are others. For example I recommend Zsh.

You can (probably) run Bash on Linux by simply opening your Terminal program.

You can find your terminal program in your main menu, like your GNOME menu or your KDE menu, similar to the start menu on Windows.

When you run your terminal it will (probably) also run Bash.

PowerShell

PowerShell 5 is a shell for Windows. It is available by default.

PowerShell 7 is better and runs on all of Windows, Mac and Linux.

Download: PowerShell 7

Compile Flags

Your compiler has hundreds, maybe even thousands of command line flags that help control how you build your programs.

You have already seen several of these in previous examples, however there are several more important flags you should familiarize yourself with.

You should familiarize yourself with flags that control things like

  • C++ Language Version
  • C++ Standard Library
  • Warning & Error Levels
  • Performance Optimization
  • Debugging
  • Unicode & Character Sets
  • Include File Search Directories
  • Library File Search Directories

The next sections will detail some of these flags for each compiler and how to use them and why.

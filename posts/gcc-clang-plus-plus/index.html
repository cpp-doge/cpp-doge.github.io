G++ And Clang++

What Is This About?

A common mistake is not running the right command when compiling a C++ program.

So Which Command Do I Use?

If you are using GCC then use the g++ command

g++ main.cpp -o main

If you are using Clang then use the clang++ command

clang++ main.cpp -o main

Why Is This A Thing?

The GNU Compiler Collection is known as GCC. Confusingly this collection of compilers also includes a program named gcc, however this is not what you want.

  • The gcc program is the C compiler, it does not compile C++ code.
  • The g++ program is the C++ compiler, it compiles C++ code.
  • If you are writing C++ with GCC then use the g++ command.

Clang has a similar situation, use clang for C code and use clang++ for C++ code.

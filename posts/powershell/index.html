PowerShell

PowerShell is a shell from Microsoft.

You should use PowerShell 7 if you are on Windows.

Download: PowerShell 7

PowerShell is greatly superior to the ancient and outdated legacy DOS batch shell named cmd.exe or simply named “Command Prompt” in your Windows Start Menu.

Add A Better Terminal

You should also use Windows Terminal. It has more and better features including tabs and integration with WSL.

Alternatively try ConEmu instead.

PowerShell Editions

There are two distinctly different versions of PowerShell known as “PowerShell Editions”.

These are NOT THE SAME

  • PowerShell Core Edition
    • Use this one. This is the new PowerShell from Microsoft. It runs on Mac, Linux and Windows.
  • PowerShell Desktop Edition
    • This is the default PowerShell that comes with Windows, it is old, it does not work well with cross platform tools like GCC, don’t use it.

Execution Policy

This is not a problem if you use PowerShell Core.

The Problem

PowerShell Desktop Edition forbids running PowerShell scripts by default.

The Simple Solution

Use PowerShell Core instead.

The Other Solution

If you must, this can instead be fixed by setting your execution policy.

See: Set-ExecutionPolicy

This may require Administrator privileges.

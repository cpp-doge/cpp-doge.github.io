What Is A Number System

What Is A Base

The base of your number system is how many digits you have.

We doggos use a beef hotdog number system.

No, wait… we use a base ten number system.

What does that actually mean though?

I’m The Only Tigger

We have been using base ten for so long in our normal lives, like at the grocery store or when talking about football scores, that it just feels natural and normal to most people.

However, different cultures in history have used different number systems, like the ancient Mayan’s supposedly used a base 60 number system! WOW!

What Are Digits

A digit, like a letter of the alphabet, is a little squiggly line symbol that people drew on rocks and paper thousands of years ago, so now we’re mostly stuck with them.

Digits represent numbers.

We have ten digits, zero through nine. These represent the numbers, you guessed it, zero through nine.

  • 0 (this is actually the first digit)
  • 1 (this is actually the second digit)
  • 2 (this is actually the third digit)
  • 3 (this is actually the fourth digit)
  • 4 (this is actually the fifth digit)
  • 5 (this is actually the sixth digit)
  • 6 (this is actually the seventh digit)
  • 7 (this is actually the eighth digit)
  • 8 (this is actually the ninth digit)
  • 9 (this is actually the tenth digit)

Or you might prefer to start at one since starting at zero can be a little bit confusing if you are not used to computer numbers yet.

It’s a little less confusing also because now the numbers and their place names line up, where 1 is first, 2 is second etc..

  • 1 (this is the first digit)
  • 2 (this is the second digit)
  • 3 (this is the third digit)
  • 4 (this is the fourth digit)
  • 5 (this is the fifth digit)
  • 6 (this is the sixth digit)
  • 7 (this is the seventh digit)
  • 8 (this is the eighth digit)
  • 9 (this is the ninth digit)
  • 0 (this is the tenth digit)

Unfortunately for you, dear doggo, is that most computers start numbering at zero. Sad Confusion Noises

The Problem & The Goal

Numbers in different base systems mean something different.

For example, the number 123 does not mean the same thing in base ten as the number 123 in base 60.

So 123 is NOT EQUAL to 123!? What!?

To solve this we need to convert between bases.

How do we convert from base to base? For example, how do we convert a number from base ten to base 60?

We seek a math formula that will allow us to do this.

Arriving at this math formula and understanding it is the goal.

Solving The Problem

Bigger Numbers

How we we talk about and write down numbers bigger than ten if we only have ten digits?

The answer is number systems. We put a bunch of digits in a row, and then we add and multiply.

Breaking It Down

Large numbers as you know them have “places”. Each place is a multiple of the base. In regular human numbers each place is a multiple of ten.

We write a number such as “five hundred thirty seven” like 537, but what does that mean?

To do more analysis, split this number into places and give them their place names.

          this is the ones place
                               |
       this is the tens place  |
                            |  |
this is the hundreds place  |  |
                         |  |  |
                         |  |  |
                         v  v  v

                         5  3  7

Why is this useful?

Understanding what this actually means and how it’s useful gets into doing some basic gradeschool arithmetic.

Adding It Up

We have chopped this number in to places. Now what?

Here Are The Rules:

  • Take each place
  • Multiply the number in that place by its place value
  • Add the results

So

  • Multiply by one for the ones place
  • Multiply by ten for the tens place
  • Multiply by one hundred for the hundreds place

Then we add all these results together to get our final number.

Breaking down this example number 537 we have

  • A 7 in the ones place
  • A 3 in the tens place
  • A 5 in the hundreds place

We multiply each number by its place

  • A 7 in the ones place, multiplied by one
  • A 3 in the tens place, multiplied by ten
  • A 5 in the hundreds place, multiplied by one hundred

Finally we add up the results.

We can write this down as a basic math equation and we get something like this.

537 = (5 * 100) + (3 * 10) + (7 * 1)

We can think of this as saying there are

  • five hundreds
  • three tens
  • seven ones
  • all added together

We can check our work to see if this is true

  • 5 * 100 = 500
  • 3 * 10 = 30
  • 7 * 1 = 7

If we add these up using a calculator

500 + 30 + 7

We get the number 537.

This may seem strange and pointless, but it will matter when we talk about the numbers that computers use, binary numbers.

Solving Binary Numbers

Binary is a base two number system. The word binary basically means two. A bicycle is named that because bi means two and a bicycle has two wheels.

Binary Digits

Unlike our familiar human numbers, binary only has two digits, zero and one. Bi means two, so we can only have two numbers in our number system.

  • 0 (this is the first digit)
  • 1 (this is the second digit)

This has a strange problem from before where the numbering and digit names are off by one. You could think about it like this instead.

  • 1 (this is the first digit)
  • 0 (this is the second digit)

But that doesn’t help much either because zero doesn’t really match to the idea of being second. Oh well.

Anyway, binary has two digits, zero and one, that’s it.

Each digit means what it does in our natural base ten system, but of course there are only two digits available.

Binary Conversion

Numbers are stored in binary in a computer. The base of the numbers used by the computer is not the same as what we normally use as humans.

If we want to know which “human” number is stored in a computer, we need to look at the binary number, and translate the binary digits into a regular normal human base ten decimal number.

To do this, we use the formula we looked at from before, except we change the base.

In human numbers our base is 10. In binary numbers, the base is two, so each place is a multiple of two.

Let’s consider a simple binary number in a computer like

1101

Well, this number does not mean one thousand, one hundred and one, because it’s a binary number in a computer.

So how do we figure out how much this actually is as a number we’re actually familiar with?

Breakdown The Places

Let’s break down the places of this number

                  this is the ones place
                                       |
               this is the twos place  |
                                    |  |
        this is the fours place  |  |  |
                                 |  |  |
    this is the eights place  |  |  |  |
                              |  |  |  |
                              v  v  v  v

                              1  1  0  1

Wait, these are different than before, why these places?

We get these places because the base is two so each place is a multiple of two.

  the ones place = 1

  the twos place = 1 * 2 = 2

 the fours place = 1 * 2 * 2 = 4

the eights place = 1 * 2 * 2 * 2 = 8

These are actually powers of the base, where the power of the first place starts at zero and increases by one for each place.

  the ones place = 2^0 = 1 (the zero power of two)

  the twos place = 2^1 = 2 (the first power of two)

 the fours place = 2^2 = 4 (the second power of two)

the eights place = 2^3 = 8 (the third power of two)

Put Humpty Back Together

Using this new perspective, we can write the equation for our binary number like this

(1 * 8) + (1 * 4) + (0 * 2) + (1 * 1)

As before, we might conceptualize this by saying out loud, verbally, that there are

  • One eights
  • One fours
  • Zero twos
  • One ones
  • all added together.

So if we do the math we get

  • 1 * 8 = 8
  • 1 * 4 = 4
  • 0 * 2 = 0
  • 1 * 1 = 1

Add these all together with a calculator and we get

8 + 4 + 1 = 13

So the binary number 1101 actually means the number thirteen as we are normally familiar with it.

INFO

notice that this is the same as

(1 * 2^3) + (1 * 2^2) + (0 * 2^1) + (1 * 2^0)

which is just expressing the places as their powers.

This may not seem super useful, but it’s convenient especially when writing a computer program to do these kinds of conversions.

This Is Bad

You may often see C++ code like this

#include <stdio.h>
#include <math.h>

int main() {
  float x = sqrt(3.14);

  printf("%f", x);
}

These are C compatibility headers. Don’t use them in C++.

Do This Instead

Put the letter c in front, remove the .h and add the std namespace.

#include <cstdio>
#include <cmath>

int main() {
  float x = std::sqrt(3.14);

  std::printf("%f", x);
}

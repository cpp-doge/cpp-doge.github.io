Don’t Be Mean

You are being mean to newbies if you tell them to use some tool because you like it, rather than considering the actual needs of the newbie and what will help them be productive and successful.

Case In Point: I like Vim but I don’t recommend that newbies use Vim because that would probably be mean.

Offer To Help

If you are not willing to offer free interactive support to newbies for this (probably advanced) thing you say they should do then you are giving bad advice. Actually you’re probably giving bad advice even if you do offer to help, because: see “Reducing Complexity”.

Case In Point: I like Vim. If I recommend that someone use Vim I then I always do so in a context where I am ready, available and willing to help them extensively and interactively to get setup and working with Vim in a quick and mostly painless way. However, as per the point about cognitive load, I don’t recommend Vim unless someone already shows an active interest AND I feel it might be appropriate given that person’s skill level.
